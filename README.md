Following the "Data Value Chain" model, the process is divided into three core stages:
Raw Data: Collection of unstructured information.
Aggregation: Grouping data into meaningful metrics (Sum, Average, Median) to identify macro trends.
Cleaning: Removing duplicates, handling null values, and correcting formats to ensure data integrity.
